# natex

Automated natural-experiment discovery: find, validate, and estimate regression
discontinuities (and, in later phases, difference-in-differences designs) in any
tabular dataset. Modern reimplementation of the LoRD3 lineage (Herlands et al.
KDD 2018; Herlands 2019; Jakubowski et al. JMLR 2023) with the corrections
documented in docs/math_audit_final.md. Work in progress.
